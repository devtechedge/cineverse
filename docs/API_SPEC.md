# Cineverse — API Specification

> The OpenAPI 3.0 schema is auto-generated by FastAPI at `/api/v1/openapi.json`
> and rendered as Swagger UI at `/docs`. This document summarises every
> endpoint with example requests/responses and the unified error envelope.

All successful responses share:

```json
{ "success": true, "data": ..., "error": null }
```

Errors:

```json
{
  "success": false,
  "data": null,
  "error": { "code": "VIDEO_NOT_FOUND", "message": "Video not found" }
}
```

| HTTP | `error.code` |
|------|--------------|
| 400 | `BAD_REQUEST` |
| 401 | `UNAUTHORIZED` |
| 403 | `FORBIDDEN` |
| 404 | `NOT_FOUND` |
| 409 | `CONFLICT` |
| 410 | `GONE` |
| 413 | `PAYLOAD_TOO_LARGE` |
| 415 | `UNSUPPORTED_MEDIA_TYPE` |
| 422 | `VALIDATION_ERROR` |
| 429 | `RATE_LIMITED` |
| 500 | `INTERNAL_ERROR` |

## Auth

### `POST /api/v1/auth/register`
```json
// req
{ "email": "you@example.com", "password": "supersecret1", "full_name": "You" }
// 201
{ "success": true,
  "data": { "access_token": "...", "refresh_token": "...", "token_type": "bearer", "expires_in": 1800 } }
```

### `POST /api/v1/auth/login`
```json
{ "email": "you@example.com", "password": "supersecret1" }
```
Returns the same shape as `/register`.

### `POST /api/v1/auth/refresh`
```json
{ "refresh_token": "..." }
// 200
{ "success": true, "data": { "access_token": "...", "token_type": "bearer", "expires_in": 1800 } }
```

### `POST /api/v1/auth/logout`
```json
{ "refresh_token": "..." }
// 200
{ "success": true, "data": { "logged_out": true } }
```

### `GET /api/v1/auth/me`
```json
{ "success": true,
  "data": { "id": 1, "email": "you@example.com", "full_name": "You",
            "is_active": true, "role": "user", "created_at": "2026-..." } }
```

## Videos

### `POST /api/v1/videos/init` — begin resumable upload
```json
{ "filename": "trip.mp4", "size_bytes": 1048576, "title": "Trip", "tags": ["travel"] }
// 201
{ "data": { "upload_id": "abc123", "chunk_size": 1048576, "expires_in": 86400 } }
```

### `POST /api/v1/videos/chunk/{upload_id}` — multipart chunk
Form fields: `chunk_index` (int), `chunk` (file). Returns current `UploadProgress`.

### `POST /api/v1/videos/finalize/{upload_id}` — assemble + enqueue ffmpeg
```json
{ "data": { "video_id": 42, "status": "processing" } }
```

### `GET /api/v1/videos` — list (paginated, searchable, filterable)
Query: `page`, `page_size`, `search`, `tag`, `has_journal`, `status`, `sort`.

### `GET /api/v1/videos/{id}`
Returns full `VideoResponse`.

### `GET /api/v1/videos/{id}/stream`
- For ready videos returns the HLS master playlist (`application/vnd.apple.mpegurl`).
- Falls back to MP4 progressive streaming with **HTTP range** support (`206 Partial Content`).

### `GET /api/v1/videos/{id}/hls/{path}`
Serves an HLS segment or rendition playlist.

### `GET /api/v1/videos/{id}/thumbnail`
JPEG, 640 px wide.

### `DELETE /api/v1/videos/{id}` — soft delete + cascade cleanup.

### WS `/api/v1/ws/upload/{upload_id}`
Server pushes `UploadProgress` messages:
```json
{ "upload_id": "abc123", "stage": "processing", "pct": 50, "message": "Transcoding to HLS" }
```

## Journal

| Verb | Path |
|------|------|
| `POST`   | `/api/v1/videos/{video_id}/journal` |
| `GET`    | `/api/v1/videos/{video_id}/journal` |
| `PUT`    | `/api/v1/journal/{entry_id}` |
| `DELETE` | `/api/v1/journal/{entry_id}` |
| `GET`    | `/api/v1/journal/search?q=…` |

Create example:
```json
{ "timestamp_seconds": 12.5,
  "content": { "type": "doc", "content": [...TipTap JSON...] },
  "content_text": "Plain text mirror for FTS." }
```

## Clips & Sharing

| Verb | Path |
|------|------|
| `POST` | `/api/v1/videos/{video_id}/clips` |
| `GET`  | `/api/v1/clips/{clip_id}` |
| `GET`  | `/api/v1/clips/{clip_id}/stream` |
| `POST` | `/api/v1/clips/{clip_id}/share` |
| `POST` | `/api/v1/videos/{video_id}/share` |
| `GET`  | `/api/v1/share/{token}` — public |
| `GET`  | `/api/v1/share/{token}/stream` — public |

Clip create:
```json
{ "title": "Best part", "start_time": 12.0, "end_time": 24.0 }
```

Share response:
```json
{ "data": { "token": "…", "url": "https://host/api/v1/share/…",
            "expires_at": "2026-…", "view_count": 0 } }
```
